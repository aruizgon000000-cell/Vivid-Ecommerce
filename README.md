# VIVID Fashion

Modern ecommerce SPA focused on premium UX, performance and clean architecture.

---

## Overview

VIVID is a modern ecommerce platform designed with a strong focus on:

- premium user experience
- performance-first architecture
- responsive design
- accessibility
- SEO optimization
- scalable frontend structure

The project was built to simulate a modern digital commerce experience using lightweight technologies and optimized frontend interactions.

---

- Responsive ecommerce interface

- Interactive shopping experience
- Optimized checkout flow
